# Quartz 스타일 변경 가이드

Quartz에서 스타일을 변경하는 방법은 크게 4가지가 있습니다.

## 1. 기본 테마 설정 (quartz.config.ts)

가장 간단한 방법으로, 색상과 폰트를 변경할 수 있습니다.

### 색상 변경

```typescript:quartz.config.ts
theme: {
  colors: {
    lightMode: {
      light: "#faf8f8",        // 페이지 배경색
      lightgray: "#e5e5e5",    // 테두리 색상
      gray: "#b8b8b8",         // 그래프 링크, 더 진한 테두리
      darkgray: "#4e4e4e",     // 본문 텍스트 색상
      dark: "#2b2b2b",         // 헤더 텍스트 및 아이콘 색상
      secondary: "#284b63",    // 링크 색상, 현재 그래프 노드
      tertiary: "#84a59d",     // 호버 상태, 방문한 그래프 노드
      highlight: "rgba(143, 159, 169, 0.15)",  // 내부 링크 배경, 강조 텍스트
      textHighlight: "#fff23688",  // 마크다운 강조 텍스트 배경
    },
    darkMode: {
      // 다크 모드 색상도 동일한 구조로 설정
    },
  },
}
```

### 폰트 변경

Google Fonts에서 제공하는 모든 폰트를 사용할 수 있습니다.

```typescript:quartz.config.ts
theme: {
  typography: {
    header: "Schibsted Grotesk",  // 헤더 폰트
    body: "Source Sans Pro",      // 본문 폰트
    code: "IBM Plex Mono",        // 코드 폰트
  },
}
```

## 2. 커스텀 스타일 (quartz/styles/custom.scss)

더 세밀한 스타일 변경이 필요할 때 사용합니다. 이 파일은 `base.scss`를 import하므로, 모든 기본 스타일을 오버라이드할 수 있습니다.

```scss:quartz/styles/custom.scss
@use "./base.scss";

// 커스텀 스타일 작성
.page {
  // 페이지 최대 너비 조정
  max-width: 1200px;
}

// 특정 컴포넌트 스타일 변경
.search {
  border-radius: 10px;
}

// CSS 변수 사용
a {
  color: var(--secondary);
  transition: all 0.3s ease;
  
  &:hover {
    color: var(--tertiary);
    transform: translateY(-2px);
  }
}
```

### 사용 가능한 CSS 변수

Quartz는 다음과 같은 CSS 변수를 제공합니다:

- `--light`: 배경색
- `--lightgray`: 연한 회색 (테두리)
- `--gray`: 회색
- `--darkgray`: 진한 회색 (본문 텍스트)
- `--dark`: 어두운 색 (헤더 텍스트)
- `--secondary`: 보조 색상 (링크)
- `--tertiary`: 3차 색상 (호버)
- `--highlight`: 강조 배경
- `--textHighlight`: 텍스트 강조 배경
- `--headerFont`: 헤더 폰트
- `--bodyFont`: 본문 폰트
- `--codeFont`: 코드 폰트

## 3. 레이아웃 브레이크포인트 (quartz/styles/variables.scss)

반응형 디자인의 브레이크포인트를 조정할 수 있습니다.

```scss:quartz/styles/variables.scss
$breakpoints: (
  mobile: 800px,   // 이 값 이하는 모바일 레이아웃
  desktop: 1200px, // 이 값 이상은 데스크톱 레이아웃
);
```

### 기타 변수

```scss:quartz/styles/variables.scss
$sidePanelWidth: 320px;  // 사이드 패널 너비
$topSpacing: 6rem;       // 상단 여백
$boldWeight: 700;        // 굵은 글씨 두께
$semiBoldWeight: 600;    // 중간 굵기
$normalWeight: 400;      // 일반 굵기
```

## 4. 컴포넌트별 스타일

각 컴포넌트는 자체 스타일 파일을 가질 수 있습니다. 예를 들어:

- `quartz/components/styles/darkmode.scss` - 다크모드 토글 버튼
- `quartz/components/styles/search.scss` - 검색 바
- `quartz/components/styles/graph.scss` - 그래프 뷰
- `quartz/components/styles/explorer.scss` - 파일 탐색기

특정 컴포넌트의 스타일을 변경하려면 해당 컴포넌트의 `.tsx` 파일을 확인하여 어떤 스타일 파일을 사용하는지 확인한 후, 해당 파일을 수정하거나 `custom.scss`에서 오버라이드할 수 있습니다.

## 실전 예제

### 예제 1: 링크에 밑줄 추가

```scss:quartz/styles/custom.scss
@use "./base.scss";

a {
  text-decoration: underline;
  text-decoration-color: var(--secondary);
  text-underline-offset: 2px;
}
```

### 예제 2: 코드 블록 스타일 변경

```scss:quartz/styles/custom.scss
@use "./base.scss";

pre {
  border-radius: 8px;
  padding: 1.5rem;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}
```

### 예제 3: 페이지 너비 조정

```scss:quartz/styles/custom.scss
@use "./base.scss";

.page {
  max-width: 1400px;  // 기본값보다 넓게
  padding: 2rem;
}
```

### 예제 4: 다크모드 전환 애니메이션

```scss:quartz/styles/custom.scss
@use "./base.scss";

body {
  transition: background-color 0.3s ease, color 0.3s ease;
}

* {
  transition: background-color 0.3s ease, color 0.3s ease, border-color 0.3s ease;
}
```

### 예제 5: 글자 크기 줄이기

```scss:quartz/styles/custom.scss
@use "./base.scss";

html {
  font-size: 14px; // 기본 16px에서 14px로 줄임
}

body {
  font-size: 0.9rem; // 본문 텍스트 크기 줄이기
}
```

## Footer에 소셜 링크 추가하기

GitHub, LinkedIn 등의 소셜 미디어 링크는 **Footer 컴포넌트**에 추가합니다. Footer는 모든 페이지 하단에 표시됩니다.

### 위치

소셜 링크는 `quartz.layout.ts` 파일의 `sharedPageComponents` 섹션에서 Footer 컴포넌트를 설정할 때 추가합니다.

### 설정 방법

```typescript:quartz.layout.ts
export const sharedPageComponents: SharedLayout = {
  head: Component.Head(),
  header: [],
  afterBody: [],
  footer: Component.Footer({
    links: {
      GitHub: "https://github.com/your-username",
      LinkedIn: "https://www.linkedin.com/in/your-profile",
      Twitter: "https://twitter.com/your-handle",
      Email: "mailto:your-email@example.com",
      // 원하는 만큼 링크 추가 가능
    },
  }),
}
```

### 현재 설정 확인

현재 프로젝트의 Footer 설정은 다음과 같습니다:

```9:14:quartz.layout.ts
  footer: Component.Footer({
    links: {
      GitHub: "https://github.com/sungmin-woo-devops",
      "Discord Community": "",
    },
  }),
```

### LinkedIn 링크 추가 예시

LinkedIn 링크를 추가하려면 `links` 객체에 항목을 추가하면 됩니다:

```typescript:quartz.layout.ts
footer: Component.Footer({
  links: {
    GitHub: "https://github.com/sungmin-woo-devops",
    LinkedIn: "https://www.linkedin.com/in/your-profile",
    "Discord Community": "", // 빈 링크는 제거하거나 URL을 추가하세요
  },
}),
```

### Footer 스타일 커스터마이징

Footer의 스타일을 변경하려면 `quartz/styles/custom.scss`에서 오버라이드할 수 있습니다:

```scss:quartz/styles/custom.scss
@use "./base.scss";

footer {
  // Footer 텍스트 정렬 변경
  text-align: center;
  
  // Footer 불투명도 조정
  opacity: 0.8;
  
  // 링크 간격 조정
  ul {
    gap: 1.5rem;
    justify-content: center; // 링크를 중앙 정렬
  }
  
  // 링크 스타일 변경
  a {
    font-weight: 600;
    transition: all 0.2s ease;
    
    &:hover {
      color: var(--tertiary);
    }
  }
}
```

### 참고사항

- Footer는 모든 페이지의 하단에 표시됩니다
- `links` 객체의 키는 링크 텍스트로 표시됩니다
- 빈 문자열(`""`)을 URL로 사용하면 링크가 비활성화됩니다
- 링크는 가로로 나열되며, 기본적으로 왼쪽 정렬됩니다

## 팁

1. **변경 사항 확인**: 스타일을 변경한 후 `npx quartz build` 또는 `npx quartz dev`로 빌드하여 변경 사항을 확인하세요.

2. **CSS 변수 활용**: 하드코딩된 색상 대신 CSS 변수를 사용하면 라이트/다크 모드 전환이 자동으로 처리됩니다.

3. **브라우저 개발자 도구**: 브라우저의 개발자 도구를 사용하여 어떤 클래스나 요소에 스타일을 적용해야 하는지 확인할 수 있습니다.

4. **점진적 변경**: 한 번에 많은 스타일을 변경하기보다는, 작은 변경부터 시작하여 점진적으로 개선하는 것이 좋습니다.
